The SCC Wizard plugin provides interactive skills to help you create, edit, validate, and understand SCC configurations without memorizing the schema.

## Prerequisites

Enable the scc-wizard plugin in your org config:

```json
{
  "defaults": {
    "enabled_plugins": [
      "scc-wizard@sandboxed-code-official"
    ]
  }
}
```

Or install standalone in Claude Code:

```bash
/plugin marketplace add CCimen/sandboxed-code-plugins
/plugin install scc-wizard@<marketplace-key>
```

## Which Wizard Do I Need?

| I want to... | Use |
|--------------|-----|
| Set up a new organization | [Org Config Wizard](/guides/wizard/org-config-wizard) |
| Create a config for my team (team-managed) | [Team Config Wizard](/guides/wizard/team-config-wizard) |
| Understand why a plugin isn't working | [Explain Wizard](/guides/wizard/explain-wizard) |
| Check if my config is valid | [Validate Wizard](/guides/wizard/validate-wizard) |
| Add teams to an existing org config | [Org Config Wizard](/guides/wizard/org-config-wizard) (Update mode) |
| Migrate org-managed teams to team-managed | [Org Config Wizard](/guides/wizard/org-config-wizard) (Migration mode) |

## Key Concepts

### Org-managed vs Team-managed

| Model | Description | When to Use |
|-------|-------------|-------------|
| **Org-managed** | Org admin controls all plugin lists | Small orgs, pilots, centralized control |
| **Team-managed** | Teams maintain their own config files | Large orgs, self-service, distributed teams |
| **Mixed** | Some teams org-managed, some team-managed | Migration, different team needs |

<Aside type="tip">
Start with **org-managed** for simplicity. Move to **team-managed** when teams need self-service plugin management.
</Aside>

### Blocked vs Denied vs Disabled

| Term | Meaning | Can Change? |
|------|---------|-------------|
| **Blocked** | Security boundary (`security.blocked_*`) | No - immutable |
| **Denied** | Delegation or allowlist rejection | Yes - update delegation |
| **Disabled** | Removed by `defaults.disabled_plugins` | Yes - re-enable |
